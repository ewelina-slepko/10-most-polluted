## Setup
To run this project, install it locally using npm:

* Clone this repository
$ git clone https://github.com/ewelina-slepko/Most-polluted-cities

* Go into the repository
$ cd Most-polluted-cities

* Install dependencies
$ npm install

* Run the app
$ npm start
